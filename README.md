# 儿童时间管理应用 - 小兔子与大灰狼的冒险

这是一个面向儿童的移动端网页应用，旨在通过趣味化方式帮助儿童建立时间观念并改善拖拉习惯。应用采用了小兔子与大灰狼的冒险主题，通过生动有趣的动画和语音提示，鼓励儿童按时完成任务。

## 功能特点

### 1. 倒计时模块
- 大字号数字显示剩余时间（格式：分:秒）
- 环形进度条实时显示剩余时间百分比
- 支持中文任务名称输入
- 简单易用的开始/暂停/重置控制按钮

### 2. 报时模块
- 剩余时间小于60%时开始语音提示
- 按20%间隔进行报时（60%、40%、20%三个节点）
- 友好、鼓励性的语音内容
- 可开关的报时功能

### 3. 趣味动画模块
- 大灰狼追逐小白兔的动态场景
- 随时间推移，小白兔状态变化（放松→紧张→慌乱）
- 提前完成任务时有胜利动画和欢快音效

### 4. 家长后台设置
- 密码保护机制（默认密码：123456）
- 可自定义倒计时时长（5-60分钟）
- 可调报时间隔
- 4个常用计时方案快捷选择
- 音效、动画效果开关

### 5. 数据统计
- 记录每次任务的详细数据
- 可视化展示任务完成情况
- 支持数据导出功能

## 使用方法

1. 在浏览器中打开 `index.html` 文件
2. 在输入框中输入任务名称（如：写作业、读书等）
3. 点击「开始」按钮启动计时器
4. 任务完成后点击「完成任务！」按钮
5. 家长可以点击右上角的设置图标（⚙️）进行高级设置
6. 点击左上角的统计图标（📊）查看历史记录

## 技术说明

- 纯前端实现，无需后端服务器
- 使用 localStorage 存储用户设置和历史记录
- 响应式设计，适配各种移动设备
- 离线使用支持

## 开发说明

如果您想进一步开发或修改此应用，可以直接编辑 `index.html` 文件。主要功能都已在单文件中实现，包括：

- HTML 结构
- CSS 样式
- JavaScript 功能逻辑

## 浏览器兼容性

- Chrome (移动端)
- Safari (iOS)
- 微信内置浏览器
- 其他主流移动浏览器

## 注意事项

- 语音报时功能需要浏览器支持 Web Speech API
- 首次使用时可能需要授予麦克风/音频权限
- 数据仅存储在本地浏览器中，请定期导出备份重要数据
- 默认家长密码为 123456，建议在正式使用时修改
